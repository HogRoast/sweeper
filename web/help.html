Sweeper data is easy to read once you know what everything stands for. So here's a quick explanation.

Please contact us if you have any further questions.

Date

Date of the match

League

Match league

Home Team

Name of the home team.

Away Team

Name of the away team.

Mark

The algo rating for the match; the more positive the number the more likely a home win.

M#

The number of historical matches at this mark.

H#

The number of historical matches with the same Sweeper rank which have resulted in home wins.

H%

The percentage of historical matches at the same Sweeper rank which resulted in a home win.

HO

Minimum home odds required to make a bet statistically viable.

D#

The number of historical matches with the same Sweeper rank which have resulted in draw wins.

D%

The percentage of historical matches at the same Sweeper rank which resulted in a draw win.

DO

Minimum draw odds required to make a bet statistically viable.

A#

The number of historical matches with the same Sweeper rank which have resulted in away wins.

A%

The percentage of historical matches at the same Sweeper rank which resulted in a away win.

AO

Minimum away odds required to make a bet statistically viable.
